# One-Piece-SOCKETS-TCP---Troca-de-Mensagens-de-Batalha-Naval-
- Python  - Sockets TCP/IP - Threading com Mutex - Sincronização de Processos
Temática: One Piece

Como instalar:


1. Baixe o repósitorio

2.Instale as dependências (Node) pelo cmd
cd local_da_pasta
npm install

3.Rodar o projeto -
npm run dev

Como jogar:
1. Abra 2 terminais. Recomendável abrir projeto pelo Visual Studio Code para iniciar os terminais
2. Crie Terminal 1 e digite: python marinha.py
3. Crie Terminal 2 e digite: python piratas.py
4. Jogue alternando terminais/turnos
